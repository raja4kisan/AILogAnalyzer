## ?? **HOW TO USE**

### **1. Analyze Logs:**
```
1. Paste logs in textarea
2. (Optional) Select log type from dropdown
3. Watch character counter update
4. Click "Analyze with AI"
5. See loading steps progress
6. View comprehensive results
7. Download report if needed
```

### **2. View History:**
```
1. Click "History" in navbar
2. Browse past analyses
3. See severity badges
4. View error/warning counts
5. Click outside to close
```

### **3. Check Stats:**
```
1. Click "Stats" in navbar
2. View total analyses
3. See errors/warnings found
4. Check severity distribution chart
5. Monitor system metrics
```

### **4. Toggle Dark Mode:**
```
1. Click sun/moon icon (top-right)
2. Theme switches instantly
3. Preference saved
4. Works in all views
```
